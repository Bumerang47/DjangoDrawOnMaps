### DjangoDrawOnMaps
Простой сервис по отрисовке географических координат из БД на карте.

Backend: **Django**

Отрисовка координат: **Leaflet.js**

СУБД: **SQLite**


Для миграции объектов модели на существующую БД с примерами:

` python manage.py migrate --fake-initial`
